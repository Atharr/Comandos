Comandos principais do git:

  • Início

    • git init (inicializa um novo repositório, cria o diretório .git)
    • ou
    • git clone <endereço do repositório em: ssh ou http> (copia o repositório remoto para um diretório local)
    • git remote add origin <endereço do repositório em: ssh ou http> (se conecta ao repositório do git/nuvem)
    • git remote -v (mostra se conectou)

  • Baixar

    • git pull origin <NomeDaBranch> (Nome da ramificação que vc quer baixar)
    • git pull (recebe os arquivos atualizados do remoto para o local)

  • Info

    • git status (mostra o status dos arquivos, alterações etc; se os arquivos no seu diretório local estão certos ou não)
    • git log (mostra o histórico dos seus commits)

  • Ramificações

    • git checkout -b <nomeDaNovaBranch> (cria e troca para uma nova ramificação)
    • git checkout <nomeDaBranch> (troca para uma ramificação existente)
    • git branch -b (mostra as ramificações que tem no repo da nuvem)
    • git branch (mostra as ramificações que tem no seu repo local)
    • git branch <nomeDaBranch> (mudar de branch)

  • Editando arquivos

    • git add <nomeDoArquivoOuDiretorio> ou . (adiciona o arquivo alterado ao seu git local; "." acrescenta todos os arquivos do diretório)
    • git commit -m "Descrição do que vc alterou" (adiciona um ponto na linha do tempo da ramificação com uma descrição das alterações; é recomendável que a cada mudança seja feito um commit)

  • Enviar

    • git push -u origin <nomeDaBranch> (envia as alterações locais na ramificação para o repositório remoto; "-u" configura a origem da ramificação no primeiro envio)
    • git push (envia as alterações locais na ramificação para o repositório remoto)
